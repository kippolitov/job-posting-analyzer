How it works

Open a job posting in Chrome, click the extension icon, and it reads the page you're viewing to extract structured details and, if you've set up a candidate profile, a fit score with a short rationale. Save postings to a personal library that follows your account across devices.

Companion web app

Sign in with the same Google account on the companion web app to browse, search, filter, sort, and compare your saved library on a bigger screen, edit your candidate profile, and analyze a job posting from an uploaded Word (.docx) or PDF document instead of a browser tab. It's the same account, the same free/Premium limits, and the same data — nothing to set up separately. Open the link from the extension's side panel.
